# Personal Portfolio

Welcome to my personal portfolio website! This is where I showcase my work, share my thoughts through blog posts, and provide information about my professional journey.

## About This Site

This portfolio serves as a central hub for my professional presence online. Here you'll find:

- **Projects**: A collection of my work and side projects
- **Blog**: Articles and thoughts on topics I'm passionate about
- **CV/Resume**: My professional experience and skills
- **Contact**: Ways to get in touch with me

## Getting Started

If you want to run this locally or fork it for your own use:

```bash
pnpm install
pnpm run dev
```

## Built With

This site is built using modern web technologies:

- **[Astro](https://astro.build)** - The web framework for content-driven websites
- **[TailwindCSS](https://tailwindcss.com/)** - For styling
- **[DaisyUI](https://daisyui.com/)** - UI component library

## Project Structure

The site is organized with components, content, and pages that make it easy to maintain and update:

- `/src/components/` - Reusable UI components
- `/src/content/blog/` - Blog posts in Markdown
- `/src/pages/` - Main pages and routes
- `/public/` - Static assets like images

## Customization

Key configuration can be found in `/src/config.ts` where you can update site title, description, and other global settings.
